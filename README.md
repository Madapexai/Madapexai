## 我相信的几件事

**管理的本质正在重构。** 从管人到管Agent，从指令到循环，从监督到自进化。这不是工具的升级，是组织范式的跃迁。

**Loop > Prompt。** 写一个好的Loop比写一百个Prompt更有价值。Loop是可复用、可进化、可审计的，Prompt只是一次性的指令。

**实战是唯一的检验标准。** 三本书不是概念推演，是从真实Agent团队运营中总结的方法论。每一个方法论都踩过坑、翻过车、验证过。
